# todo-react-native

## Clone & Install

 * Clone this repo git clone https://github.com/Koteswari24/todo-react-native.git
 * **cd ToDoList**
 * Run **npm install**
 * Run **react-native start** to open node.js
 * Run **react-native run-android** for Android platform
